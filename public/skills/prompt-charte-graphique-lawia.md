## RÈGLES CANVA — Configuration de la marque

### Couleurs à enregistrer dans "Brand Kit"
| Nom | Hex |
|-----|-----|
| Bleu principal | #6A9BCC |
| Bleu foncé | #4A7BA8 |
| Bleu clair | #A8C4E0 |
| Bleu pâle | #D3E5F8 |
| Noir | #141413 |
| Code (noir léger) | #1A1917 |
| Crème | #FAF9F5 |
| Gris | #B0AEA5 |
| Gris clair | #E8E6DC |
| Terre cuite (négatif) | #C4745B |

### Polices à utiliser dans Canva
| Rôle | Police Canva |
|------|-------------|
| Titres | **Georgia** (disponible nativement) |
| Corps | **Calibri** ou **Gill Sans** |
| Code/technique | **Courier New** ou **Source Code Pro** |
| Manuscrit | **Caveat** (disponible dans Canva) |

### Éléments graphiques à réutiliser
- Bordures noires épaisses (3px minimum visuellement)
- Ombres offset géométriques (pas de blur, décalage de 4-8px)
- Barres décoratives (48px × 3px) comme séparateurs
- Tags : fond noir, texte crème, uppercase, petit
- Icônes : style stroke uniquement, jamais fill, en bleu clair sur fond sombre

### Interdits absolus
- Pas de dégradés
- Pas de rouge, vert, orange, violet
- Pas de photos de blessures ou de sang
- Pas d'images violentes ou choquantes
- Pas de polices fantaisie (hors Caveat pour accroches)
- Pas d'ombres diffuses (uniquement offset géométriques)
- Pas de coins carrés sur les grands conteneurs (toujours 12px)
