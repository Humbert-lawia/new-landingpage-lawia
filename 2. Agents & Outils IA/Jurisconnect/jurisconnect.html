Legifrance 13 outils +
fl_articles_code+
Recherche des articles dans les 73 codes juridiques francais. Par mots-cles ("contrat de travail") ou par numero d'article ("L1234-5"). Filtrage par champ, tri par pertinence ou date.
fl_liste_codes+
Retourne la liste complete des 73 codes en vigueur disponibles : Code civil, Code penal, Code du travail, Code de commerce, etc.
fl_juris_judiciaire+
Recherche dans les decisions de la Cour de cassation, des cours d'appel et des tribunaux judiciaires. Filtrage par juridiction, publication au bulletin, mode panorama ou texte integral.
fl_juris_administrative+
Recherche dans les decisions du Conseil d'Etat, des cours administratives d'appel et des tribunaux administratifs. Pagination et tri par date ou pertinence.
fl_juris_financiere+
Recherche dans les decisions de la Cour des comptes, des chambres regionales des comptes et de la Cour de discipline budgetaire et financiere.
fl_decisions_cnil+
Recherche dans les deliberations de la CNIL : sanctions, avis, mises en demeure, recommandations, rappels a l'ordre. Filtrage par nature de deliberation.
fl_decisions_qpc+
Recherche dans les decisions du Conseil constitutionnel : QPC, DC (controle a priori), L (declassement). Filtrage par numero de decision.
fl_conventions+
Recherche dans les conventions collectives nationales (base KALI). Par nom, numero IDCC ou mots-cles. Mode panorama pour vue d'ensemble.
fl_textes_consolides+
Recherche dans les textes legaux consolides (LODA) : lois, decrets, ordonnances. Possibilite de cibler un texte precis par son numero.
fl_journal_officiel+
Recherche dans le Journal Officiel. Filtrage par date de publication, ministere, emetteur, type de texte (loi, decret, arrete).
fl_dernier_jo+
Recupere le sommaire du ou des derniers JO publies (1 a 10). Ideal pour la veille juridique quotidienne.
fl_emetteurs_jo+
Liste toutes les autorites emettrices pour filtrer les recherches JORF : ministeres, autorites independantes, etc.
fl_types_textes_jo+
Liste tous les types de textes publies au JO : loi, decret, arrete, avis, circulaire, etc.
BOFiP — Doctrine fiscale 2 outils +
fl_bofip+
Recherche dans le Bulletin Officiel des Finances Publiques. Doctrine fiscale officielle de la DGFiP : impots, TVA, plus-values, fiscalite des entreprises.
fl_bofip_document+
Recupere le texte integral d'un document BOFiP a partir de son identifiant. Contenu complet avec references et liens.
Judilibre — Cour de cassation 3 outils +
fl_judilibre+
Recherche dans la base Judilibre de la Cour de cassation. Acces a toutes les decisions publiees en open data par la haute juridiction judiciaire.
fl_judilibre_decision+
Recupere le texte integral d'une decision Judilibre a partir de son identifiant. Motifs, dispositif, visa, references.
fl_judilibre_taxonomie+
Explore la taxonomie Judilibre : chambres, matieres, formations. Utile pour connaitre les categories de classement disponibles.
RNE / INPI / BODACC — Entreprises 9 outils +
fl_entreprises+
Recherche d'entreprises dans le Registre National des Entreprises par nom, SIREN ou mots-cles. Pagination integree.
fl_fiche_entreprise+
Fiche complete d'une entreprise : denomination, forme juridique, dirigeants, adresse, activite, immatriculations.
fl_annonces_bodacc+
Recherche dans les annonces legales BODACC : creations, modifications, radiations, procedures collectives (redressement, liquidation).
fl_acte_entreprise+
Telecharge un acte juridique depose : statuts constitutifs, PV d'assemblee generale, actes de cession, etc.
fl_bilan_entreprise+
Telecharge un bilan comptable depose par une entreprise (comptes annuels).
fl_extrait_rne+
Telecharge un extrait du Registre National des Entreprises, equivalent fonctionnel du Kbis.
fl_fichiers_temp+
Liste les fichiers temporaires en attente de telechargement (actes, bilans, extraits).
fl_supprimer_fichier+
Supprime un fichier temporaire specifique du cache local.
fl_purger_fichiers+
Purge tous les fichiers temporaires pour liberer l'espace disque.
EUR-Lex — Droit europeen 9 outils +
fl_ue_recherche+
Recherche de textes europeens par titre ou mots-cles : reglements, directives, decisions de l'UE.
fl_ue_par_celex+
Recherche un texte europeen par son numero CELEX. Ex : 32016R0679 pour le RGPD. Acces direct.
fl_ue_article+
Recupere le texte d'un article specifique d'un document europeen. Ex : article 5 du RGPD.
fl_ue_sommaire+
Liste tous les articles d'un document europeen. Utile pour connaitre la structure d'un reglement ou d'une directive.
fl_ue_considerant+
Recupere un considerant specifique. Les considerants exposent les motifs et le contexte du texte.
fl_ue_considerants+
Liste tous les considerants d'un document europeen.
fl_ue_recherche_dans+
Recherche en texte integral a l'interieur d'un document europeen. Trouve un passage precis dans un long reglement.
fl_ue_infos+
Metadonnees d'un document : titre officiel, date d'adoption, auteur institutionnel, base juridique, entree en vigueur.
fl_ue_texte_integral+
Recupere le texte integral complet d'un document europeen.
Systeme — Diagnostic 2 outils +
fl_diagnostic+
Verifie la connexion a toutes les API et retourne l'etat de chaque service : actif, cle manquante, erreur. Outil de depannage.
fl_boss+
Retourne les informations du Bulletin Officiel de la Securite Sociale. Doctrine officielle en matiere de protection sociale.
Installation

5 minutes. Aucune
competence technique.

1. Obtenez vos cles

# JurisConnect (Legifrance, RNE, EUR-Lex)
https://auth.openlegi.fr

# Judilibre (Cour de cassation)
https://piste.gouv.fr

# BOFiP : aucune cle requise

2. Installez uv

# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

3. Configurez Claude Desktop

// claude_desktop_config.json
{
  "mcpServers": {
    "jurisconnect": {
      "command": "uv",
      "args": ["run", "--python", "3.12",
        "--directory", "/chemin/vers/MCP JurisConnect",
        "python", "server.py"],
      "env": {
        "JURISCONNECT_TOKEN": "votre_token",
        "JUDILIBRE_KEY": "votre_cle_piste"
      }
    }
  }
}

4. Relancez Claude

# C'est fait. Testez avec :

"Quel est le texte actuel
  de l'article 1240 du Code civil ?"

"Recherche les dernieres sanctions
  CNIL sur le RGPD"

"Doctrine fiscale BOFiP
  sur les plus-values immobilieres"
